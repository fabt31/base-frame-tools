# Getting Started with Base Frame Tools

## Prerequisites
- Node.js 18+
- A Farcaster account
- Base wallet with ETH for gas

## Create Your First Mint Frame

```bash
npm run dev
```

Visit `http://localhost:3000/api/frame` and paste the URL into the Warpcast Frame Validator.

## Deploy to Vercel
```bash
vercel deploy --prod
```

## Frame Validator
Test your frames at: https://warpcast.com/~/developers/frames
